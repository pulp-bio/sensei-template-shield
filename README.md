# SENSEI Template Shield

This repository contains the hardware design files for the **SENSEI Template Shield**, a template for sensei sensing shields.

## Overview

The **SENSEI Template Shield** simplifies the development process by providing a ready-to-use PCB template. The template includes correctly placed and routed board-to-board connectors compatible with the SENSEI Base Board, enabling straightforward integration and easy stacking of additional sensing shields.
